Проверка валидности номера телефона - Краткое руководство
Этот проект представляет собой простую форму для проверки правильности введенного номера телефона. 
Для этого используется технология AJAX на стороне клиента и PHP на стороне сервера. 
В процессе валидации осуществляется проверка кода страны, длины номера и префикса оператора.

Инструкции по использованию
Введите международный номер телефона в следующем формате:

"+" в начале,
код страны (например, +373, +380),
код оператора (68, 777),
и сам номер (014195, 20355).
Пример ввода:
+37362015186
В случае возникновения ошибки, система выдаст сообщение, указывающее на конкретное место ошибки. 
Также имеется база кодов для более чем 200 стран.
